HTML（超文本标记语言）

	1.用来描述网页结构

	2.不是编程语言，而是一种标记语言

	3.标记语言是一套标记标签

	4.我们学习HTML主要是学习HTML标签

HTML基本架构

<html>

    <head>
   
        <title>我的第一个页面</title>

    </head>

    <body>

           键盘敲烂 月薪过万   

    </body>

</html>

	html标签：根标签

	head标签：头标签

	title标签：网页标题标签

	body标签：内容标签

	注：标签书写推荐用小写字母！！！

HTML里标签的两种形式

双标签：

	语法：<html> 我是内容 </html>

单标签：

	语法：<img src="" alt="">

HTML标签关系：

	嵌套关系（父子关系）：

	<head>

		<title>我是儿子</title>

	</head>

	并列关系（兄弟关系）：

	<head>我是熊大</head>

	<body>我是熊二</body>

声明：

	<!DOCTYPE html>

	! HTML5

	DOC == document 文档

	TYPE == type 类型

	html 超文本标记语言

	文档类型是超文本标记语言

注：声明必须要写在第一行！！！

编码字符集：

	<meta charset="UTF-8">

	utf-8 万国码 识别地球上所有的文字

HTML标签语义化

	1.方便代码的阅读和维护。

	2.同时让浏览器或是网络爬虫可以很好地解析，从而更好分析其中的内容。

	3.使用语义化标签会具有更好地搜索引擎优化。

HTML标签：

	1.标题标签

	<h1></h1>-<h6></h6>

	标题小结：（1）加了标题的文字会变的加粗，字号也会依次变大（2）一行是只能放一个标题的

	2.段落标签

	<p> 文本内容 </p>

	注：文本在一个段落中会根据浏览器窗口的大小进行自动换行。

	3.水平线标签

	<hr>

	注：单标签 颜色是灰色的

	4.换行标签

	<br>

	注：在代码页里直接敲回车是没有效果的！！！所以br标签实现强制换行 ！！！

	5.盒子标签（重点）

	<div> 内容 </div>

	<span> 内容 </span>

	注：div和span没有语义化 用来布局。

	区别：div独占一行 而span不会独占一行。

文本格式化标签（认识）

	1.文字以粗体方式显示：

	<b> 内容 </b>

	<strong> 内容 </strong>

	2.文字以斜体方式显示：

	<i> 内容 </i>

	<em> 内容 </em>

	3.文字添加删除线：

	<s> 内容 </s>

	<del> 内容 </del>

	4.文字添加下划线：

	<u> 内容 </u>

	<ins> 内容 </ins>

注：这些标签认识就可以（淘汰标签）。

标签属性

	语法：

	<标签名 属性1="属性值1"> 内容 </标签名>

图片标签：

	语法：

	<img src="图片路径">

	属性：

	src：图片路径

	alt：图片加载失败时提示文本

	title：鼠标悬停图片时提示信息

	width：图片宽度

	height：图片高度

	border：图片边框粗细

	注：

	1.图片标签是单标签。

	2.属性键值对 key = “value”；

	3.标签可以拥有多个属性，必须写在开始标签中，位于标签名后面。

	4.属性之间不分先后顺序，标签名与属性、属性与属性之间均以空格分开。

超链接标签：

	语法：

	<a href="链接路径"> 文本或图像 </a>

	属性：

	href：URL地址

	target：链接页面打开方式

		属性值：

		_self当前页打开

		_blank新打开一个页面

	注：

	1.外部链接需要书写协议 http://或https://

	2.如果链接为空 需要在href里书写#

	3.不仅文本可以创建超链接，图片，表格，音频，视频等元素都可以创建超链接。

注释标签：

	语法：

	<!-- 注释内容 -->

	注：

	1.注释不会被浏览器所识别。

	2.注释的主要功能是给程序员看的。

相对路径：

	以引用文件之网页所在位置为参考基础，而建立出的目录路径。因此，当保存于不同目录的网页引用同一个文件时

，所使用的路径将不相同，故称之为相对路径。

	同一级路径：

	只需要输入文件名称即可。

	下一级路径：

	输入文件夹名称及文件名称。

	例如：<img src="images/baidu.gif">

	上一级路径：

	在文件名之前输入../ 表示向上一个层级

	如果是上两级 那么就是../../

	例如：<img src="../baidu.gif" />

绝对路径：

	绝对路径以Web站点根目录为参考基础的目录路径。

之所以称为绝对，意指当所有网页引用同一个文件时，所使用

的路径都是一样的。

	例如：“D:\web\img\logo.gif”

		"http://www.jd.com" 

	注：

	它的写法 特别是符号 \ 并不是 相对路径的 /。

锚点定位：

	超链接通过id值实现当前页面跳转。

base标签：

	<base target="_blank">

	1.base 可以设置整体链接的打开状态。

	2.base 写到head之间。

	3.把所有的连接 都默认添加 target="_blank"。

pre标签：

	预格式化文本标签

	标签 元素中的文本通常会保留空格和换行符。
	
	而文本也会呈现为等宽字体。

	<pre> 内容 </pre>

特殊字符代码：

	空格： &nbsp;

	小于号： &lt;

	大于号： &gt;

	总结：

	1.是以运算符&开头,以分号运算符;结尾。

	2.他们不是标签，而是符号。

	3.HTML中不能使用小于号和大于号特殊字符，浏览器会将它们作为标签解析。
